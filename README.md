# loremrc

## What is this.

Simple lorem ipsum generator


##  install 


```bash
npm i loremrc

```

## init 


```javascript

const lorem = new (require('initrc')).base();

```


## Get a word 


```javascript

let word = lorem.word();

```
## Get a stence


```javascript

let stence = lorem.stence();

```

## Get a paragraph


```javascript

let stence = lorem.paragraph();

```
